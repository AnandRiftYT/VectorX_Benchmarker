# 🚀 VectorX GPU Benchmark
**Developed by Anand**

VectorX is a high-performance, lightweight GPU benchmarking tool built with **Electron** and **WebGL**. Designed to push both legacy and modern silicon to their absolute limits, VectorX specializes in "Hardware Handshaking" to extract maximum performance from dedicated GPUs.


* **Cross-Platform Logic:** Optimized for Linux and Windows.
* **Low-Level Handshake:** Custom WebGL rendering pipeline designed.
