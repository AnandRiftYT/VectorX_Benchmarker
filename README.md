# VectorX_Benchmarker
# 🚀 VectorX GPU Benchmark
**Developed by Anand 

VectorX is a high-performance, lightweight GPU benchmarking tool built with **Electron** and **WebGL**. Designed to push both legacy and modern silicon to their absolute limits, VectorX specializes in "Hardware Handshaking" to extract maximum performance from dedicated GPUs.


* **Cross-Platform Logic:** Optimized for Linux (Mint/Zorin) and Windows.
* **Low-Level Handshake:** Custom WebGL rendering pipeline designed to bypass "Amoeba" driver bottlenecks.


### 🛠️ Installation
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/VectorX_Benchmarker.git`
2. Install dependencies: `npm install`
3. Run the Benchmark: `npm start`
